## Краткое описание
Программа предназначена для удобной работы с репозиторием материалов для монтажа видео (или для других задач, где требуется организация большого количества файлов).

Возможности:
- Создание репозитория медиафайлов
- Добавление и удаление кастомных тегов
- Назначение и удаление тегов для каждого файла
- Назначение и удаление описания файлов
- Отображение информации по конкретному файлу в окне превью
- Поиск файлов по названию, описанию и тегам
- Drag'n'drop напрямую из окна программы в окно видеоредактора (протестировано на Adobe Premiere Pro)

- ffmpeg.exe рядом с .exe-файлом программы
- набор иконок для программы
Зависимости поставляются вместе с программой в [релизах](#загрузить-программу) (в RAR-архиве).

## Первый запуск
При первом запуске программы нужно выбрать папку для репозитория при помощи кнопки "Выбрать папку". Сейчас поддерживаются только файлы внутри одной отдельно взятой папки (без иерархии папок внутри).

Форматы файлов должны совпадать с теми, которые [поддерживает программа](#поддерживаемые-форматы-файлов). 

После корректного выбора папки программа:
- Генерирует превью для файлов
- Добавляет информацию о файлах в базу данных
- Автоматически назначает файлам стандартные теги
  - `Audio` для аудиофайлов
  - `Video` для видеофайлов
  - `Image` для изображений

Превью создаются в отдельной подпапке внутри репозитория.

База данных создается внутри рабочей директории программы.

## Поддерживаемые форматы файлов
Форматы, с которыми работает программа, задаются в fhandler.py в следующих константах:
```python
ALLOWED_IMAGE_FORMATS = [".jpeg", ".jpg", ".png", ".gif", ".bmp", ".tiff", ".jfif"]
ALLOWED_VIDEO_FORMATS = [
    ".mp4",
    ".mkv",
    ".avi",
    ".mov",
    ".webm",
    ".flv",
    ".wmv",
    ".mpeg",
    ".mpg",
    ".m4v",
]
ALLOWED_AUDIO_FORMATS = [
    ".mp3",
    ".wav",
    ".flac",
    ".aac",
    ".ogg",
    ".opus",
    ".m4a",
    ".wma",
    ".aiff",
    ".alac",
]
```
